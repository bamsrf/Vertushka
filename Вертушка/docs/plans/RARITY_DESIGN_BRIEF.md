## PROMPT (отдай дизайнеру целиком)

Спроектируй визуальную подсветку «редких пластинок» в мобильном виниловом приложении (React Native, Expo, тёмная тема).

КЛЮЧЕВОЙ ПРИНЦИП: НЕ бейджи-пилюли, а целая карточка релиза становится сигналом редкости. Цвет, свечение и анимация привлекают внимание сами по себе. Карточка должна выглядеть «премиально», а не «помечено стикером».

В приложенных файлах:
- `theme.ts` — токены текущей темы (точные цвета, шрифты, радиусы). Опирайся на них.
- `RecordCard.tsx` — текущая реализация карточки в режиме списка. Сохрани существующие пропорции и информационную плотность, добавь поверх неё подсветку.
- `color_palette.jpeg`, `design_bar.jpg` — общая дизайн-система проекта.
- `current_screens/` — скриншоты живого приложения, чтобы понимать контекст скролла и плотность.

ТРИ ТИРА РЕДКОСТИ:

1. FIRST PRESS — Heritage gold
- Палитра: champagne → amber → bronze (#F4D27A → #B8860B → #6B4423)
- Aura (свечение вокруг карточки): тонкий shimmer бежит против часовой стрелки по периметру за ~8 сек
- Доп. эффект: раз в 10 сек диагональный световой блик пробегает по обложке (как блик на пластинке под лампой)
- Иконка-маркер (правый верхний угол обложки, без подложки, с цветным glow): корона (Lucide Crown)
- Текстовый лейбл в метаданных (просто текст в цвет тира, без фона): «1-й пресс»
- Настроение: благородный, спокойный, винтажно-ценный

2. LIMITED — Cold platinum violet
- Палитра: silver → violet → indigo (#C0C0D8 → #6B4DCE → #2A1F4E)
- Aura: pulse opacity 0.5 → 0.9 → 0.5 за 4 сек, бесконечно. Без бега по периметру — статично-премиально
- Доп. эффектов нет (намеренно сдержанный)
- Иконка: gem (Lucide Gem)
- Лейбл: «Лимитка»
- Настроение: скульптурный, холодный премиум

3. HOT — Hot ember
- Палитра: amber → coral → deep red (#FFB347 → #FF5E3A → #B22222)
- Aura: pulse opacity 0.4 → 1.0 → 0.4 за 2 сек. Самый интенсивный
- Доп. эффект: по краям обложки еле заметная heat-haze (статичный размытый красный halo)
- Иконка: flame (Lucide Flame)
- Лейбл: «HOT» (латиницей, набран в caps)
- Настроение: горячий, желанный, «многие хотят это сейчас»

КРИТИЧЕСКИ ВАЖНО — три тира должны быть видны на одном экране и НЕ выглядеть как дискотека. Разные ЧАСТОТЫ анимаций сами создают иерархию: первопресс блестит спокойно, лимитка дышит, HOT пульсирует.

ДВА ВАРИАНТА ВЁРСТКИ КАРТОЧКИ — оба обязательны:

**GRID (плитка)** — вертикальная карточка, квадратная обложка ~150×150 сверху + 1–2 строки текста (название, артист) снизу:
- Aura охватывает всю карточку, glow читается как ореол вокруг
- Иконка в правом верхнем углу обложки
- Текстовый лейбл — отдельной строкой между названием и метаданными ИЛИ в строке метаданных

**LIST (список)** — горизонтальная карточка, обложка слева ~80×80 + блок текста справа (название, артист, год, формат, цена):
- Aura охватывает всю горизонтальную полосу
- Дополнительно: цветная ЛЕВАЯ КРОМКА 3px (тот же градиент тира) — экономит вертикальное место и работает как акцент при плотном вертикальном скролле в коллекции
- Иконка в правом верхнем углу обложки
- Текстовый лейбл — в правом блоке, в строке метаданных рядом с годом и форматом

ДЕТАЛЬНАЯ СТРАНИЦА РЕЛИЗА — отдельно от карточки:
- Секция «Особенности» над секцией цен
- Для каждого применимого тира — крупный «карточный» элемент: цветной glow + иконка 24px + название тира + одна строка описания («Канонический первый пресс мастер-релиза» / «Специальное издание» / «Высокий спрос на Discogs»)
- Все три тира могут быть видны одновременно (вертикальный список)

ЧТО НУЖНО ПОКАЗАТЬ:

1. Все три тира в варианте GRID — одна обложка, разный тир-эффект (3 мокапа)
2. Все три тира в варианте LIST — одна обложка, разный тир-эффект (3 мокапа)
3. Экран коллекции в режиме LIST: ~6 карточек в скролле, из них 1 first_press, 1 limited, остальные обычные. Чтобы понять, как соседство «горящих» и обычных карточек выглядит вместе. (HOT в коллекции скрыт — это by design.)
4. Экран поиска в режиме GRID: 6–9 карточек, из них 1 first_press, 1 limited, 1 HOT. Чтобы оценить три тира рядом.
5. Секция «Особенности» на детальной странице с тремя крупными тир-блоками одновременно.
6. Раскадровка анимации для каждого тира (4–6 кадров с timing) — отдельно aura, отдельно доп. эффект (если есть).

ТЕХНИЧЕСКИЕ ОГРАНИЧЕНИЯ:
- React Native + Reanimated 3 (анимации на UI-thread)
- Glow через нативный shadow + анимированный opacity (не Skia, не BlurView — производительность критична для скролла)
- expo-linear-gradient для градиентов
- В grid одновременно может скроллиться 30–50 карточек, в list — 20–30
- Иконки из lucide-react-native
- Не предлагай эффекты, которые требуют web-only API (CSS filters, backdrop-filter blur и т.п.)
